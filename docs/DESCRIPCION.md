# Descripcion funcional

IGNITE LABORATORIO implementa el flujo Inicio -> Aprender -> Evaluacion -> Resultados -> Certificado/Reporte.

La informacion se guarda localmente en el navegador mediante almacenamiento local. El certificado se genera a partir de la plantilla oficial incluida en public/assets/certificate-template.png y el logotipo oficial se conserva en public/assets/ignite.png.
